    • DatePicker


    jQueryUI provides datepicker() method that creates datepicker UI for an input. If a date is chosen, feedback is shown as the input's value. You can customize the date format and language, restrict the selectable date ranges and add in buttons and other navigation options easily. I used the datepicker for the purchase date and repair date selection in the form and addded validation like maxDate to restrict future date selections.

    "search" feature


    I added a search feature on the FAQ page using the filter() method. The search function searches all questions in a JSON array that match a specified key words entered by the user and the results have been displayed right below the search button with all other questions after it.

    Ajax/JSON


    I used a JSON file to store the faq questions and answers and i use Ajax method to retrive the data from the JSON file and display it on a new window when the user clicks the FAQ button.

    File upload feature


    I added a file upload option on the form in the repair details section using the input type=file. The files property returns a FileList object, representing the file or files selected with the file upload button. Through the FileList object, you can get the name, size and the contents of the files.
